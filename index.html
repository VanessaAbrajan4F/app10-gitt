HTML5

Elementos de Bloque:

Un elemento en bloque es aquel que contiene un bloque significativo de contenido que debería mostrarse en su propia línea. Por ejemplo: párrafos, tablas, listas, etc…

Formulario:

Un formulario permite a un usuario ingresar datos. La información que se ingresa se manda a un servidor en un grupo de datos que consisten del nombre de los elementos y sus valores. Para hacer uso de un formulario, utilizaremos las etiquetas: Requiere de los atributos action y method, los cuales son la URL del form-handler y el método HTTP por el cual se envían los datos, respectivamente.

Css

Cascading Style Sheets (CSS) ‘Hoja de estilo en cascada’ es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML. CSS separa la estructura de un documento de su presentación. La información de estilo puede ser definida en un documento separado o en el mismo documento HTML.

Js

2do parcial

jQuery (la librería open-source en Javascript) te permite amaestrar el poder de JavaScript para cumplir una multitud de cosas increíbles en tu página web. Con jQuery, puedes: añadir, borrar o modificar elementos HTML en tu página, cambiar el estilo de elementos en tu página modificando el CSS asociado, animar elementos en tu página, enviar y recibir datos de un servidor vía AJAX (asynchronous JavaScript and XML) de modo que tu página no tenga que cagarse de nuevo tras enviar un formulario.

JS

Los archivos .js guardan funciones y variables globales que se ejecutarán en la página web pudiendo llamar a sus funciones desde cualquier subpágina sin tener que incrustar scripts en cada una de ellas y ahorrando así código. Para quien tenga idea de construcción de páginas web, podría decirse que es similar a un archivo CSS pero en lugar de clases y estilos este archivo genera scripts.
